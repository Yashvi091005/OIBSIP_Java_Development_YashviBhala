## 📌 Task 2: Number Guessing Game (GUI-Based)

### 🔹 Description
A Java-based Number Guessing Game developed using **Java Swing**.  
The application generates a random number between 1 and 100, and the user attempts to guess it. The system provides feedback and calculates a score based on the number of attempts.

### 🔹 Features
- GUI-based interface using Swing
- Random number generation
- User input handling
- Attempt counter
- Score calculation
- Reset functionality

### 🔹 Technologies Used
- Java
- Java Swing
- AWT

## 📌 Task 3: ATM Interface (GUI + Database)

### 🔹 Description
A GUI-based ATM Interface developed using **Java Swing** and **MS Access Database**.  
The system allows users to log in securely and perform basic banking operations with transaction data stored persistently.

### 🔹 Features
- Secure login system
- Deposit and withdrawal operations
- Balance update
- Transaction history
- Database connectivity using JDBC

### 🔹 Technologies Used
- Java
- Java Swing
- JDBC
- MS Access Database
